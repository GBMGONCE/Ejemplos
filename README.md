# Ejemplos

en la hoja .gitattributes se está diciendo que detecte automáticamente los archivos de texto y normalice sus saltos de línea a LF al subirlos al repositorio.

en la hoja analog.py el código mide repetidamente el nivel de voltaje en el Pin 36, lo convierte en un número digital (entre 0 y 4095) y lo muestra por pantalla cada 0.1 segundos, ademas como hay un potenciometro conectado el numero cambiara cuando se mueva la perilla.

En la hoja leds.py se esta prendiendo y apagando 4 leds conectados a los pines 12, 18, 19 y 13 los cuales se apagan despues de un segundo.

en la hoja libmotor.py este codigo contorla dos motores usando pines digitales, y define funciones para mover el carro en diferentes direcciones encendiendo/apagando los pines correctos.

en la hoja motores.py este codigo dice que el carro debe avanzar durante 2 segundos y luego se detiene.

en la hoja olde.py El código inicializa la pantalla OLED por I2C y muestra dos líneas de texto en ella.

en la hoja rgbneopixel.py El código configura una tira de 3 NeoPixels y enciende cada uno con un color.
